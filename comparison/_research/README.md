# Comparison Research Notes

This directory is for narrative market synthesis that is useful context for the comparison program.

It is intentionally **not canonical source-of-truth data**.

Allowed here:

- market landscape summaries
- category narratives
- positioning notes
- trend observations

Not allowed here:

- structured competitor facts that belong in canonical records
- hand-maintained comparison tables that duplicate canonical data
- website filter or table source data

If a note here starts acting like source of truth, move the structured data into competitor canon or remove the note.
